# ML Test (Node/Express - React)

ML Test requires [Node.js](https://nodejs.org/) to run.

## Backend Installation 

Install the dependencies and devDependencies and start the server.

```sh
cd mercado-backend
npm i
node run start
```

## Frontend Installation 

Install the dependencies and devDependencies and start the server.

```sh
cd mercado-frontend
npm i
node run start
```
